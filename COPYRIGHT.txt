This folder contains the database described in the article:

    DeepToF: Off-the-Shelf Real-Time Correction of Multipath Interference in Time-of-Flight Imaging
    Julio Marco, Quercus Hernandez, Adolfo Muñoz, Yue Dong, Adrian Jarabo, Min H. Kim, Xin Tong, Diego Gutierrez
    ACM Transactions on Graphics, Vol.36(6)
  
and generated with the rendering framework described in the article:

    A Framework for Transient Rendering
    Adrian Jarabo, Julio Marco, Adolfo Muñoz, Raul Buisan, Wojciech Jarosz, Diego Gutierrez
    ACM Transactions on Graphics, Vol.33(6)
